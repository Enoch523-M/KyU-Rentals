# listing.html - Property Detail Page

**Purpose:** Shows the full details of a single rental property when a student clicks on a listing card.

---

## What This Page Does

1. Reads the `?id=X` parameter from the URL to know which property to display
2. Finds that property in the `listings` array from data.js
3. Renders a full **photo gallery** with clickable thumbnails
4. Shows **price, location, distance**, and vacancy status
5. Lists all **amenities** with icons
6. Displays **landlord contact information** with a call button

---

## Structure Breakdown

### Minimal HTML
```html
<div class="detail-page" id="detailPage"></div>
```

This page has **almost no HTML**. Just one empty `<div>`. Everything inside it is generated by JavaScript (`initDetailPage()` function in app.js). This is a common pattern in modern web development called **client-side rendering**.

### Why So Little HTML?

Because the content depends on WHICH property the user clicked. The page doesn't know what to show until JavaScript reads the URL parameter:

```
User clicks card for "Sunset View Apartments" (id=1)
    --> Browser navigates to listing.html?id=1
    --> JavaScript reads: id = 1
    --> JavaScript finds listing with id 1 in the data array
    --> JavaScript builds all the HTML and inserts it into the page
```

## How URL Parameters Work

When you visit `listing.html?id=3`:

```javascript
// JavaScript reads the URL
const params = new URLSearchParams(window.location.search);
// window.location.search = "?id=3"

const id = Number(params.get('id'));
// params.get('id') returns "3" (as string)
// Number("3") converts it to the number 3

const listing = listings.find(l => l.id === id);
// .find() searches the array for an object where id === 3
// Returns the matching listing object
```

## Photo Gallery

The gallery uses a simple pattern:
- One **large main image** at the top
- **Small thumbnail images** below it
- Clicking a thumbnail calls `changePhoto()` which:
  1. Changes the `src` of the main image
  2. Removes `active` class from all thumbnails
  3. Adds `active` class to the clicked thumbnail

```javascript
function changePhoto(src, thumbEl) {
  document.getElementById('mainPhoto').src = src;           // Change main image
  document.querySelectorAll('.gallery-thumbs img')
    .forEach(img => img.classList.remove('active'));          // Remove active from all
  thumbEl.classList.add('active');                            // Add active to clicked one
}
```

## Error Handling

If someone visits `listing.html?id=999` (an ID that doesn't exist):

```javascript
if (!listing) {
  detailPage.innerHTML = `
    <div class="no-results">
      <h3>Listing not found</h3>
      <p>This property may no longer be available.</p>
    </div>
  `;
  return;   // Stop here, don't try to render the rest
}
```
